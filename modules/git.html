Wat is versiebeheer?

In deze module leer je alles over versiebeheer, je kunt versie beheer gebruiken om backups te hebbem van je codeer projecten, gemakkelijk samen werken aan een project en nog veel meer.

Je zal meerdere versiebeheer programma's zoals Git en Subversion leren, en welke voordelen deze programma's hebben en waarom ze zo handig zijn.
