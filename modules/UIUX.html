Module - UI/UX

Wat zijn UI en UX?

UI & UX staan voor User Interface en User Experience. In deze module leer je alles over het maken van een design dat er niet alleen goed uitziet, maar ook prettig is om te gebruiken.

Voor UI maak je gebruik van wireframes om snel een ontwerp voor je website te maken. Je leert over kleur, structuur en andere elementen die je kunt gebruiken om jouw website duidelijk en mooi te maken.

Voor UX leer je over de kleine details die een website het juiste gevoel geven, en hoe populaire websites hun de gevoelens van bezoekers be√Ønvloeden.

Waarom is dit belangerijk?

Als je een design maakt voor een klant is het belangrijk dat de klant begrijpt wat het plan is en mogelijke veranderingen kan voorstellen voordat je de pagina gaat maken.

Bij het maken van een website is het belangrijk om semantische elementen te gebruiken in je HTML-code en om gebruik te maken van ID's en classes voor het stylen met CSS.

Ook leer je handige tips over kleur, tekst en opmaakkeuzes en hoe ze de pagina een bepaald gevoel kunnen geven.

Goed gebruik van elementen zorgt ervoor dat je website SEO-geoptimaliseerd is. Dit betekent dat zoekmachines zoals Google jouw website makkelijker kunnen vinden en voorstellen.

UI/UX is vooral het werk van een Front-end Developer, die zich meest bezighoudt met hoe de website eruitziet en aanvoelt voor de gebruiker.
