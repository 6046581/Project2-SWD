Het beroepenveld

Wat doet een software developer?

  • App development
  • Game development
  • Web development
  • Software engineer
  • Backend developer / front-end developer
  • Database administrator
  • UX/UI development
  • Cyber security

Deze beroepen hebben allemaal iets gemeen. Ze gebruiken allemaal code, dus kun je ze gaan doen na opleiding een software developer.

Er zijn tientallen programeertalen om uit te kiezen, dus er valt altijd iets te leren!

Salaris

Software developer is een goed betalend beroep:

  • Een software developer met een MBO-diploma begint meestal met een salaris tussen de €2.500 en €3.000 bruto per maand.
  • Enkele jaren ervaring kan het salaris stijgen naar €3.000 tot €4.000 bruto per maand.
  • Op hoog niveau en met veel ervaring kan het salaris oplopen tot €4.000 tot €5.500 bruto per maand, of zelfs meer in speciale rollen of bij grote bedrijven.
