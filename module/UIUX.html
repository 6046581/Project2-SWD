Module - UI/UX

UI & UX staan voor User Interface en User Experience, bij deze module leer je alles over het maken van een design dat er niet allen goed uitziet, maar ook goed voelt om te gebruiken.

Voor UI je gebruik maken van wireframes om snel een ontwerp voor je website te maken, je leert over kleur, stuctuur, en andere dingen die je kunt gebruiken om jouw website duidelijk en mooi te maken.

Wanneer je een design aan het maken bent voor een klant is het belangerijk dat de klant snapt wat het plan is en mogelijke veranderingen kan voorstellen voordat je de pagina gaat maken

Bij het maken van een website is het belangrijk om semantische elementen te gebruiken in je HTML code, en om gebruik te maken van ID's en classes voor het stylen met CSS. Correct gebruik van deze elementen zorgt ervoor dat je website SEO geoptimaliseerd is, dit beketend dat browsers zoals Google jouw website makkelijker kunnen vinden en voorstellen.
